# OpenClaw 官方首页

这是OpenClaw项目的官方首页静态页面项目。

## 特性
- 响应式设计，支持移动端和桌面端
- 现代化UI设计，渐变色主题
- 轻量无依赖，纯HTML/CSS实现
- 快速加载，性能优秀

## 本地运行
直接用浏览器打开以下页面即可查看效果：

- `index.html`：官网首页
- `admin-login.html`：后台登录页
- `admin.html`：后台仪表盘
- `content-management.html`：内容管理页
- `user-management.html`：用户管理页
- `settings.html`：系统设置页

## 部署
可以直接部署到GitHub Pages、Vercel、Netlify等静态托管平台。

## 技术栈
- HTML5
- CSS3 (Flexbox/Grid)
- 原生JavaScript (无框架依赖)
